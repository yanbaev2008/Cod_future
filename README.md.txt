## 📖 Описание
Графическое приложение для учёта личных расходов. Позволяет добавлять транзакции, фильтровать их по категории и периоду, автоматически подсчитывать сумму за выбранный промежуток времени, а также сохранять и загружать данные в формате JSON.

## 🛠 Технологии
- Python 3.x
- tkinter (встроенная GUI-библиотека)
- json, datetime (стандартные модули)

## 🚀 Как использовать
1. Убедитесь, что установлен Python 3.6+
2. Запустите приложение:
   ```bash
   python expense_tracker.py
